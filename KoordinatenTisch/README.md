# Koordinaten-Tisch Steuerung

Flask-basierte Steuerungssoftware für einen Koordinatentisch mit **GRBL (Arduino UNO)**, Kamera-Anbindung und G-Code-Verarbeitung.

---

## Installation & Start (Windows)

### 1 Virtuelle Umgebung aktivieren

```powershell
.venv\Scripts\activate
```

### 2 Abhängigkeiten installieren

```powershell
pip install -r requirements.txt
```

### 3 Anwendung starten
```powershell
python app.py
```
